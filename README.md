### Github 연동하기

1. 소스제어   
2. Git clone
    - [보기] > [명령 팔레트] 
    ```
    git clone
    ```
    - git url 입력

3. Git 설정
   ```
   git config --global user.name [username]
   git config --global user.email [email]
   ```
4. 커밋하기 (COMMIT)
    - 소스제어 > 변경 사항 > + (변경 내용 스테이징)
    - ✔ (커밋) > 메시지 입력

5. 푸시하기 (PUSH)
    - 소스제어 > … (더보기) > 푸시

6. 풀하기 (PULL)
    - 소스제어 > … (더보기) > 풀
