### Github 연동하기

1. 소스제어   
2. 레포지토리 초기화
3. Git remote 설정
    - [git URL] : 생성한 레포지토리 URL 입니다:)
    ```
    git remote add origin [git URL]
    ```
4. 커밋하기 (COMMIT)
    - 소스제어 > 변경 사항 > + (변경 내용 스테이징)
    - ✔ (커밋) > 메시지 입력

5. 푸시하기 (PUSH)
    - 소스제어 > … (더보기) > 푸시

6. 풀하기 (PULL)
    - 소스제어 > … (더보기) > 풀
